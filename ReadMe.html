Description

This plugin will allow new users to sign into your forum through the Google OAuth authentication system. This Plugin Will also Allow you to Associate/Disassociate an existing forum account to a Google Account, and Import your Profile Photo from Google.

Step 1: Upload the Files

Log in to your Website FTP server via your preferred File Transfer Protocol Client and upload the contents of the /upload/ directory to the root of Your IPB directory.
If you have customized your admin directory, upload the contents of /upload/admin/ to that directory instead.

Step 2: Import the Hook

Log in to your IP.Board Admin CP and visit the System tab -> Applications & Modules tab -> Manage Hooks page.
In the 'Install Hook' section, browse to the XML file included with these instructions and click Open.

Step 3: Configure Your Web Application And Settings:

You *must* set up a Google Web Application. You can do that here
You need no services, go directly to the API Access tab and click Create New Web Application.
The Redirect URI's *must* include {board_url}/interface/board/google.php AND {board_url}/interface/board/linkgoogle.php
You *must* also provide the resulting Google Client ID and Google Client Secret for the Web Application you make.
You *should* also configure the Google "Real Name/Display Name" and Google Default member group.
All of these Settings are Available from your IP.Board Admin CP System tab -> Applications & Modules tab -> Community Enhancements tab -> Sign In with Google page.

Step 4: Install and enable the Login Module

Log in to your IP.Board Admin CP and visit the System tab -> Tools & Settingstab -> Login Management page.
In the 'Other Available Login Handlers' section, Click the green button on the right of 'Google'.
Then, click the Red Badge On the Right of 'Google'.
If attempting to authorize to Google results in the authorization page refreshing, please drag the Google Login Module to the very *top* of the list of login modules, then clear your browser cache and attempt again.
